# QA API Tests – Postman

Projeto de testes de API REST utilizando o Postman, com foco em testes manuais e automatizados de endpoints REST.

## Objetivo
Demonstrar conhecimento em testes de API, criação de cenários positivos e negativos, automação básica e documentação.

## Tecnologias
- Postman
- API REST
- JSON

## Endpoints testados
- GET /posts
- POST /posts
- PUT /posts/{id}
- DELETE /posts/{id}

## Tipos de testes
- Funcionais
- Positivos e negativos
- Validação de status code
- Validação de estrutura e tipos de dados

## API utilizada
JSONPlaceholder (API mock para testes)

## Como executar
1. Importar a collection no Postman
2. Executar os requests individualmente ou em sequência
